Here you can prove that you paid the receiver with the Private Transaction Key (which you can obtain from
			the wallet), the transaction ID and the receiver's address.

		
Outputs that belong to receiver's address will be highlighted.

		
Private transaction key is a one time key that is automatically generated for every transaction.

		
To get it in simplewallet from CLI Suite use command get_tx_key, in GUI you can find it in the
			transaction details dialog.


		
You can also check that payment was made to your address in given transaction with your Private View Key.
			Alternatively the Tracking Key from GUI wallet can be used.
